# SistemaNotasUsuario

El proyecto consiste en un programa para guardar notas por usuario, con persistencia de datos a través de archivos

## Funcionalidades:
1. Crear una nota y añadirle contenido
2. Modificar líneas concretas de la nota
3. Añadir al final de la nota
4. Mostrar las notas
5. Mostrar una nota concreta
6. Persistencia de las notas
